 Modeling and Simulation Tools

Our most humble welcome to what is to become a full-fledged modeling and simulation environment: a modeler, a simulator, a Modelator.

Admittedly it will take some time to get to where we want to be but we feel we're off to a great start with our first offering of a statistics calculator.

While there may not be anything too exciting about a series of calculators, what's different here is how they're implemented and what's to come.

It's an entirely stable User Interface (UI) written in common HTML, CSS, and JavaScript. The backend is written in C++ and Node.js, to help ensure the most efficient processing. These calculator components are actually just smaller parts of the components that will ultimately make up Modelator.
